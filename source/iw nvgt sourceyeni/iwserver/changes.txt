0.4.9.4
Added builder commands: builder_read, builder_append, builder_delete_line, builder_edit_line, builder_write_map, builder_delete_map.
All builder commands require dev status. All write operations automatically reload the map and re-send to connected players.
Added reload_map function to re-parse map files and push updated data to players on the affected map.

0.4.9.2
Added weapon attachment system with attach and detach commands.
Replaced per-weapon suppressor booleans with generic suppressed_weapons array on player class.
Suppressor sound logic now checks the suppressed_weapons array instead of individual flags.
Removed revolver and obsolete weapon-specific suppressor toggle code from use handler.

0.4.9.1
Added credits currency. Saved to player accounts. Awarded on kills and at milestones (5 kills, 10 kills).
Added lobby_credits command for clients to query their balance.
Added C4 explosive system with remote detonation via HyperComp.
Added HyperComp commands: bunker_status, c4_count, detonate_c4, toggle_monitor, toggle_alarm, query_bunkers.
Added bunker alarm system. Added bunker monitor loop for intruder detection alerts.
HyperComp auto-awarded when a player captures their second bunker.
Bunker status now includes coordinates for each claimed bunker.
Monitor mode and alarms are now per-bunker with toggle_monitor and toggle_alarm accepting a bunker index.
Added query_bunkers command returning structured per-bunker data (index, name, coordinates, monitor/alarm status).
Fixed self-kill credit exploit: players no longer earn credits or kill count from dying to their own explosives.

0.48
The /changes command response is now displayed in a standard audio form on the client instead of raw spoken text.

0.460
Removed currentmap system. Modes now load their own maps directly. Login no longer sends map data.
Removed /currentmap command, mapupdate, select_random_map, and currentmap.txt.

0.459
Fixed Battlegrounds loading battleroyale.map instead of main.map.
Fixed lobby_wait_for_map discarding spawn coordinates from the server.

0.4.1
Revamped account experience. Registration is now a single form. First-time welcome dialog added.
Login with nonexistent account suggests registration. Connect without account prompts to register.
Account Management menu only visible when logged in. Auth code option hidden when not needed.
Fixed legacy preferences bypass. Fixed Shift+A, Shift+H, Shift+R key press consumption bugs.
Game statistics, account statistics, and mode statistics now display in proper forms.
Fixed H key to show health from server, then stamina and effects in sequence.

0.4
Added 15 new weapons: Desert Eagle, MP5, UZI, M4A1 Carbine, SCAR-H, RPK LMG, M249 SAW, AWP Sniper, Dragunov SVD, SPAS-12, MP7, FN FAL, Crossbow, Katana, Baseball Bat.
Added fire mode system. Press Shift+A to cycle between semi-automatic, burst, and full automatic on supported weapons.
Added stamina system. Stamina degrades when using melee weapons, jumping excessively, or moving with heavy weapons. Restores every 5 seconds naturally.
Added energetic potions in 4 tiers: red, blue, green, and rainbow. Each restores increasing amounts of stamina.
Added buffs and debuffs system. Health potions give Residual Healing (HP restore every 15s for 2 min). Ultra potions give stronger healing (every 25s for 5 min). Smoke bombs apply Blinded debuff.
Added equipment system with body slot. Bomb vests are now equipped as body equipment and can be removed in the equipment menu.
Replaced grave accent game menu with escape menu. New menu includes: changes, rules, account info, developers, game statistics, equipment, account statistics, mode statistics, and exit game.
Press H to see health, stamina, and active buffs/debuffs.
Updated starting loadout pools with all new weapons.

0.35
Added developer suite accessible with Shift+D (dev builds only), featuring a give all items option.
Added documentation system with help entries in iwserver/docs. Press Shift+H to browse all help topics, or Alt+Enter on an inventory item to view its help entry.
Enter key now draws weapons from inventory in addition to using items, matching quickbar behavior.
Added Shift+I to view all weapons currently in quickbar slots.
Added inventory category cycling with J and K keys. Categories: All, Weapons, Ammo, Consumables, Equipment. Tab and Shift+Tab only show items in the selected category.
