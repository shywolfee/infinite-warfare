Developer Commands

These commands are only available to developers (running from source).

/reboot - Reboot the server. All players are notified before shutdown.
/kick [name] - Kick a player from the server (also available to admins). Cannot kick developers.
/go [x y z] - Teleport yourself to the given coordinates.
/go [name x y z] - Teleport another player to the given coordinates.
/ban [name] - Ban a player by computer ID (also available to admins). Cannot ban developers.
/newadmin [name] - Add a player to the admin list.
/notify [message] - Broadcast a notification message to all players.
/motd [message] - Set the server message of the day.
/give [name item amount] - Give items to a player.
/getmyshit - Teleport all world items to your location.
/removeitems - Remove all items from the world.
/getfeedback - Read player feedback submissions.
/bugs - View all bug reports, flag them, and attach comments (also available to admins). Flags: Pending, Erroneous, Known, Closed.
