Game Modes

Infinite Warfare currently has two game modes, both accessible from the Modes tab in the lobby.

Battlegrounds: Fight for survival on a war-torn island. A fast-paced combat map set over water.

Habitat Alpha: Fight for survival in an abandoned alien colony. A large sci-fi map with domes, tunnels, airlocks, and surface paths.

Each mode loads a different map. Select a mode from the lobby and you will be deployed directly to that map with a fresh loadout.
