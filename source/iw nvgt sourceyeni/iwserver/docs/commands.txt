Commands

Press / (slash) to open the chat input. Type a message and press Enter to send it. Messages starting with / are treated as commands.

Player commands:
/bunkers - Show bunker locations on the Archipelago map.
/duomate [name] - Check a player's duo mate.
/d [message] - Send a private message to your duo mate.
/pm [name] [message] - Send a private message to another player.
/feedback [message] - Submit feedback to the developers.
/changes - View the changelog.

You can also open the command menu with Alt+C to run any command without typing it manually.
